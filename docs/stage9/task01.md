# 阶段 9 / 任务 01：阶段边界与任务总览

## 目标

阶段 9 实现摄像头 AIOP 报警联动门禁常闭窗口。服务接收海康 SDK 回调中的 `0x4021 / COMM_UPLOAD_AIOP_VIDEO`，识别来源摄像头，按 `CameraAlarmDoorInterlock.Mappings` 找到目标门禁设备和门号，在窗口开始时投递门禁常闭任务，窗口结束后投递恢复普通受控状态任务。

阶段 9 是后续阶段能力，不改变阶段 1-8 的首期验收边界。

## 硬性边界

| 边界 | 要求 |
| --- | --- |
| 默认关闭 | `CameraAlarmDoorInterlock.Enabled` 默认为 false。 |
| 不改 gRPC 契约 | 不新增、不修改外部 gRPC 方法。 |
| 不改现有表结构 | 阶段 9 默认不新增表，不修改任何现有表结构。 |
| 配置驱动 | 摄像头到门禁目标的映射来自 `Configuration/appsettings.json`。 |
| 命中来源即触发 | 收到 `0x4021` 且来源命中配置摄像头即触发，不按 AIOP JSON `type` 过滤。 |
| JSON 解析失败不阻断 | AIOP JSON 解析失败仍触发联动，只记录解析错误。 |
| 设备通道化 | 门禁常闭和恢复必须通过 `DeviceSdkDispatcher` 调用 `NET_DVR_ControlGateway`。 |
| 窗口不续期 | 同一摄像头窗口内重复报警不刷新结束时间。 |
| 停止要恢复 | 服务停止时对活动门目标执行 best-effort 恢复。 |

## 模块划分

| 模块 | 职责 |
| --- | --- |
| `CameraDoorInterlockService` | 阶段 9 总入口，管理配置、窗口、门目标状态和生命周期。 |
| `AiopAlarmEventRouter` | 从 SDK 原始回调识别 `0x4021` 并反查摄像头。 |
| `AiopVideoPayloadParser` | 按 `NET_AIOP_VIDEO_HEAD` 拆 JSON 和 JPEG 摘要。 |
| `InterlockMappingResolver` | 将摄像头来源映射到门禁设备和门号。 |
| `CameraAlarmWindowManager` | 管理每个摄像头独立报警窗口。 |
| `DoorTargetStateManager` | 管理每个门目标的活动摄像头集合和恢复条件。 |
| `DoorControlTaskFactory` | 创建 `ControlGateway` 常闭/恢复设备任务。 |

## 主流程

| 步骤 | 动作 |
| --- | --- |
| 1 | 配置启用阶段 9，并加载摄像头到门禁目标映射。 |
| 2 | 摄像头设备完成 SDK 登录和报警布防。 |
| 3 | SDK 回调收到 `0x4021 / COMM_UPLOAD_AIOP_VIDEO`。 |
| 4 | 识别来源摄像头，命中配置则继续。 |
| 5 | 尝试解析 AIOP JSON 和图片摘要；失败只记录日志。 |
| 6 | 摄像头无活动窗口时开启 `now + WindowSeconds`。 |
| 7 | 对该摄像头映射的全部门目标投递常闭任务。 |
| 8 | 窗口内重复报警只记日志，不刷新窗口，不重复常闭。 |
| 9 | 摄像头窗口结束后，从门目标活动集合移除该摄像头。 |
| 10 | 若门目标无其他活动摄像头，投递恢复普通受控状态任务。 |
| 11 | 恢复失败按配置延迟重试。 |
| 12 | 服务停止时对活动门目标 best-effort 恢复。 |

## 关键语义

| 项目 | 规则 |
| --- | --- |
| 常闭含义 | 按“禁止入门/门禁常闭”实现。 |
| 恢复含义 | 窗口结束恢复为普通受控状态，不恢复为常开。 |
| 窗口归属 | 每个摄像头独立窗口。 |
| 门目标归属 | 一个门目标可被多个摄像头同时影响。 |
| 恢复条件 | 最后一个影响该门目标的摄像头窗口结束。 |

## 阶段完成标准

| 标准 | 说明 |
| --- | --- |
| AIOP 可识别 | `0x4021` 回调可识别并定位摄像头。 |
| 映射可执行 | 一个摄像头可映射一个或多个门禁目标。 |
| 常闭可投递 | 命中报警后通过设备通道投递常闭任务。 |
| 窗口不续期 | 重复报警不刷新窗口。 |
| 共享门正确 | 多摄像头共享同一门时最后窗口结束才恢复。 |
| 恢复可重试 | 恢复失败按配置延迟重试。 |
| 停止可清理 | 服务停止时 best-effort 恢复活动门目标。 |
